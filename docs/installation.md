# 導入方法 (Installation)

TinyDataTableは、Unity Package Manager (UPM) を使用してインストールできます。

## 🛠 インストール手順

1. Unityエディタを開き、メニューの **Window > Package Manager** を選択します。
2. 左上の **「+」ボタン** をクリックし、**「Add package from git URL...」** を選択します。
3. 以下のURLを入力し、**Add** ボタンを押します。

```text

https://github.com/sakana3/TinyDataTable.git#upm

```

## 💡ご利用の前に

TinyDataTableはAddressablesでの利用を推奨しています。
必須ではありませんが同時に導入する事をお勧めします。
