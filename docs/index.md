# TinyDataTable

Unity用の軽量で高速なデータテーブル（Excel/CSV/JSONなどからデータを読み込む）管理パッケージです。

## 🌟 特徴

- **超軽量＆高速:** ランタイムでのメモリ消費を最小限に抑え、爆速でデータを検索・参照できます。
- **直感的なAPI:** わずか数行のコードでデータにアクセス可能。
- **UPM対応:** Unity Package Managerを使って簡単に導入・アップデートができます。

## 📖 クイックスタート

まずは[導入方法](installation.md)に従ってパッケージをUnityにインストールしてください。
その後、[基本的な使い方](usage.md)を読んで、最初のデータテーブルを作成してみましょう！
