# TinyDataTable

TinyDataTableはUnrealEngineのDataTableのような軽量で高速なデータテーブルを提供するアセットです。

UnityではゲームデータをScriptableObjectで扱いますが大量のボイラーコードを生みだします。  
ちょっとしたテーブルを作るのが面倒くさくてソースコードに埋め込んだままになっていたりしませんか？  
プロジェクト終盤にあちこちに散らばった大量のScriptableObjectコードにウンザリした事はありませんか？  
頑張って実装したデータテーブルがどこにあるかわからなくなり無駄な時間を費やした事はありませんか？  

TinyDataTableはそんなあなたを退屈なDataTable管理から解放します。  

## 特徴

- **導入は簡単:** スキーマーの設計からテーブルの入力までノンコードで作業できます。
- **軽量＆高速:** スキーマーとレコードはC#で出力されます。データはScriptableObjectに格納され特別なシリアライズもなしです。
- **柔軟なフィールド:** Unityで扱える全ての型を扱う事ができます。あなたのタイトル固有のデータ型も扱えます
- **UIDによる堅牢な管理:** データはUIDで管理されデータの並び順を変更してもインデクスがズレません
- **Indexerによる簡単アクセス:** レコードデータには専用のIndexer(ID)でアクセスできます。文字列とはもうサヨウナラです。
